# my-cli
exploring golang with cobra cli

# setup
After clone the repo, build the cli and to execute the cli as per the below instructions:

### to build the cli
```
$ go build -o my-cli main.go
```

## to execute the cli on terminal
```
$ ./my-cli --help
$ ./my-cli -m "kishor"
$ ./my-cli -m "kishor" -i 
```
